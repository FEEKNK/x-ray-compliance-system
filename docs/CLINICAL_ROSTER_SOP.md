# มาตรฐานการจัดเวรและการเช็คเครื่อง (Clinical Roster SOP)
*แผนก Imaging โรงพยาบาลกรุงเทพสิริโรจน์*

⚠️ **แจ้งเตือนสำคัญ:** 
ข้อมูลแบบฟอร์มและการจัดเวรในปัจจุบัน **ยังไม่เป็นทางการ (Unofficial)** โครงสร้างด้านล่างเป็นเพียงโครงร่างพื้นฐาน หัวหน้างาน (Admin) สามารถปรับเปลี่ยนแบบฟอร์มและการจัดเวร (Schedule) ได้เองอย่างอิสระผ่านระบบ Admin Dashboard เพื่อให้ตรงกับการปฏิบัติงานจริง

## 1. ช่วงเวลาการเข้าเวร (Shifts)
*   **เวรเช้า (Morning):** 08:00 - 16:00 น.
*   **เวรบ่าย (Afternoon):** 16:00 - 00:00 น.
*   **เวรดึก (Night):** 00:00 - 08:00 น.

## 2. รายละเอียดงานรายวัน (Daily Duties - โครงร่างเบื้องต้น)

### กิจกรรมพิเศษ (Special - ตรวจเช็คก่อน 08:00 น.)
*   **พนักงานหมายเลข 3 (ASSISTANT):**
    *   ตรวจเช็คเครื่อง PORTABLE
    *   ตรวจเช็คเครื่อง C-ARM

### เวรเช้า (Morning Shift - พนักงาน 4 คน)
*   **คนที 1:** Check list ยา, เวชภัณฑ์, ของ Sterile
*   **คนที 2:** Check list เครื่องอัลตร้าซาวด์ 1, 2, 3
*   **คนที 3:** เตรียมความพร้อมอุปกรณ์สำนักงาน, เอกสารต่างๆ, สำรวจพื้นที่รอคอย/ห้องเปลี่ยนเสื้อผ้า, ความพร้อมของชุดผู้ป่วย
*   **คนที 4 (พยาบาล):** ตรวจเช็คอุปกรณ์ช่วยชีวิต (Life-saving equipment)

### เวรบ่าย (Afternoon Shift - พนักงาน 4 คน)
*   **กลุ่ม TECH:**
    *   เช็คแรงดันออกซิเจน และแรงดันระบบสุญญากาศ
    *   เช็คอุณหภูมิ/ความชื้น ห้อง CT และ Mammogram
    *   ทบทวน Special case, CT case (การคีย์/เขียนใบเวชภัณฑ์ และ Consent Form)
*   **กลุ่ม ASSISTANT:**
    *   เช็คอุณหภูมิ/ความชื้น ตู้เก็บยาและเวชภัณฑ์
    *   เช็คอุปกรณ์ช่วยชีวิต
    *   บันทึกเคสเอกซเรย์กระดูกเด็ก (<15 ปี) ลงในสมุดเวรดึก
*   **ร่วมกัน (TECH & Asst):**
    *   ตรวจเช็คถฉุกเฉิน (Crash Cart)

### เวรดึก (Night Shift - พนักงาน 2 คน)
*   **TECH (หมายเลข 1):**
    *   ตรวจเช็คเครื่อง CT SCAN
    *   ตรวจเช็คอุปกรณ์ช่วยชีวิต
    *   เช็คแรงดันออกซิเจน และแรงดันระบบสุญญากาศ
*   **TECH (หมายเลข 2):**
    *   ตรวจเช็คเครื่อง MAMMOGRAM
    *   ตรวจเช็คเครื่อง BONE DENSITOMETRY
    *   ตรวจเช็คเครื่อง FLUOROSCOPY
*   **ASSISTANT:**
    *   ตรวจเช็คเครื่อง X-Ray 1, 2, 3
    *   เช็คอุณหภูมิ/ความชื้น ตู้เก็บยาและเวชภัณฑ์
    *   ตรวจเช็คอุปกรณ์ช่วยชีวิต
*   **ร่วมกัน (TECH & Asst):**
    *   ตรวจเช็ครถฉุกเฉิน (Crash Cart)

## 3. งานปฏิบัติประจำสัปดาห์ (Weekly Duties - โครงร่างเบื้องต้น)
*   **TECH:**
    *   Calibrate เครื่อง CT ตามรอบ
    *   Calibrate เครื่อง Mammogram
    *   ปิดระบบเครื่อง CT ทั้งหมด (Shut down)
*   **ASSISTANT:**
    *   ถอดเช็ดทำความสะอาดเครื่อง Injector

## 4. ระบบการแจ้งเตือนและการปรับเปลี่ยน
*   ระบบสามารถติดตามงานที่ค้าง (Pending) ตามกะที่ได้รับมอบหมาย
*   การมอบหมายงานทั้งหมดในรายการด้านบน สามารถถูกปรับแต่ง เพิ่ม ลบ หรือสร้างขึ้นใหม่ได้อย่างสมบูรณ์ ผ่านเมนู **"จัดการแบบฟอร์ม"** และ **"จัดตารางงาน"** ในหน้า Admin
